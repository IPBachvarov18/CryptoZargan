Singl rules

In the singleplayer mode of the game you play against the computer - a random number generates and you have to guess it to progress further. There are 3 difficulties.
  • Easy
  • Medium
  • Hard
There are also 2 tasks:
  • First task – On the first task the numbers cannot be repeated
  • Second task – On the second task the numbers can be repeated.
You have 13 attempts to guess the number, otherwise you lose.

Multiplayer rules

In the multiplayer mode of the game 2 people can play against each other.
One person creates a room and gives the code that was given to him so that the other one can join it.
The person who creates the room can choose his role. There are 2 different roles:
  • German – If you choose to be German you create the code. Your task is the code to be hard so that the British cannot defeat you easy.
  • British – Or if you choose to be the British you are guessing the code that was created by the German. You have 13 attempts to guess the code.
There are also 2 tasks:
  • First task – On the first task the numbers cannot be repeated
  • Second task – On the second task the numbers can be repeated.
